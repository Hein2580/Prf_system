🚑 EMS PRF SYSTEM - SETUP INSTRUCTIONS
==========================================

❌ ISSUE: Desktop icon installation doesn't work when opening files directly

✅ SOLUTION: Run the app through a local web server

📋 QUICK START:
--------------
1. Double-click "start-server.bat" 
2. Choose option 1 or 2 (depending on what you have installed)
3. Your browser will open to http://localhost:8000
4. Now the "Add to Desktop" button will work properly!

🔧 DETAILED OPTIONS:
-------------------

Option 1 - Python Server (Recommended):
• Most Windows 10/11 systems have Python
• Double-click start-server.bat → Choose "1"
• Or manually: Open Command Prompt → Run "python -m http.server 8000"

Option 2 - Node.js Server:
• If you have Node.js installed
• Double-click start-server.bat → Choose "2" 
• Or manually: Run "npx http-server -p 8000"

Option 3 - VS Code Live Server:
• Install VS Code + "Live Server" extension
• Right-click index.html → "Open with Live Server"

Option 4 - Chrome Developer Mode:
• Close all Chrome windows
• Run: chrome.exe --disable-web-security --user-data-dir="c:/temp/chrome"
• Then open index.html

🎯 WHAT THIS FIXES:
------------------
✅ Desktop icon installation will work
✅ PWA features enabled
✅ Service worker functions properly  
✅ Offline mode available
✅ No more CORS errors

🔗 AFTER SETUP:
---------------
• Visit: http://localhost:8000
• Click "Add to Desktop" button
• App icon will be created on your desktop!
• You can then access the app directly from the icon

💡 TIP: Bookmark http://localhost:8000 for easy access!
